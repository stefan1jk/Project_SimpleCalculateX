1. การบวกเลขพื้นฐาน

Expected Result: 8

2. การลบเลขพื้นฐาน
Input: 9 – 4 =
Expected Result: 5

3. การคูณเลขพื้นฐาน
Input: 6 × 7 =
Expected Result: 42

4. การหารเลขพื้นฐาน
Input: 8 ÷ 2 =

5. การหารด้วยศูนย์
Input: 9 ÷ 0 =

6. การคำนวณทศนิยม

Input: 5 + 3 × 2 =
Expected Result: (ขึ้นกับ Logic)
แบบกดตามลำดับ: (5+3)=8, 8×2 = 16

Input: 7 + 2 → AC
Expected Result: หน้าจอกลับเป็น 0 และล้างค่าทั้งหมด
