## Technology Stack Justification
1.HTML (HyperText Markup Language)
-ใช้ในการสร้างโครงสร้าง (Structure) ของหน้าเว็บ
-เหมาะกับงานประเภท UI-based ที่เน้นปุ่ม ตัวเลข และ Layout แบบ Grid

2.CSS (Cascading Style Sheets)
-ใช้กำหนดรูปแบบการแสดงผล (UI Styling) เช่น สี ปุ่ม มุมโค้ง ช่องว่าง
-เหมาะกับเว็บแอปที่ต้องการความสวยงามและประสบการณ์ใช้งานง่าย

3.JavaScript
-ใช้ในการประมวลผล Logic ของเครื่องคิดเลข เช่น +, -, ×, ÷
-ทำงานภายใน Browser โดยตรง ไม่ต้องใช้ Server
-ตอบสนองแบบ Real-time ทำให้เหมาะกับ Calculator ที่ต้องการอัปเดตทันที
